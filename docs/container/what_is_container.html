仮想マシン（VM）

仮想マシンとは，専用の仮想化ソフトウェアによって部物理マシン上に作成され，仮想コンピュータシステムとして機能する仮想環境である．VMを使うことで，複数の異なるOSを1台のコンピュータ上で同時に実行可能となる．これにより，計算機資源を柔軟に管理することが可能である．
VMは，物理マシンと同様に独自のOS，ストレージ，ネットワーク等を備えており，同じホスト上で実行されている他のVMから完全に分離されている．
